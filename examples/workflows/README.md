# Prerequisites

## Install Pi

```bash
npm install -g @earendil-works/pi-coding-agent
```

## Install Kiro provider and dependencies

```bash
pi install npm:pi-provider-kiro
npm install -g @earendil-works/pi-tui @earendil-works/pi-ai
```

## Install MCP for Pi

```bash
pi install npm:pi-mcp-adapter
```
